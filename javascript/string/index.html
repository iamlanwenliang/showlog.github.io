字符串的操作方法

在这篇里面会详细的讲到字符串相关的用法，注意事项等等

一、定义

字符串就是用单引号或者双引号括起来的字符，其中的字符可以是任何字符， 声明有两种方式：
var str = new String();
(效果等于str = {})这种方式严格的来说也可以，但是平时不建议使用，因为使用new出来的数据类型是object，但是这种声明出来的是一个类似于数组的字符串，将字符串按照length分割开来分别存在对象里面了。 Var str = ‘’;
这是常用的字符串声明方式，这种声明出来数据类型就是string的，也是比较正常的使用方式；
①　如果单引号中还包括单引号或者双引号里面还包括双引号，其中包涵的单引号或者双引号也是字符内容的话，一般可以用反斜杠（\）来表示，位置在特殊符号前面。这里要提醒的一点是字符串里面如果是ascII编码的话，在输出时浏览器会自动解析为相对应的字符。
②　这里介绍常用的两个小技巧，不过这是es6标准，如果你的浏览器不支持的话这里就需要更新到支持es6的最新版或者在支持es6的浏览器上调试。第一个就是怎么让字符串换行呢？我们一般第一想法都是在想要换行的地方添加\n就可以换行了，今天在这里介绍的是另一种es6标准里面的，用反引号括起来，反引号在键盘的esc键的下方第一个，字符串用反引号括起来，你在书写的时候字符串里面有几个换行，在显示时就显示几个换行
例如：`你的
我的` 那么字符串输出的时候也就是这样输出的。
第二个是字符串拼接，这个我相信在平时使用中大家是很常用到的，基本上大家现在的方式都是使用加号（+）把多个字符串连起来，今天介绍的是es6新增的一种模板字符串，
例如：var man = ‘那个男人’;
var using = ‘看起来’;
var strong = ‘很强壮’;
var talk = ‘${man}${using}${strong}’;
这样就拼起来了，是不是很方便呢

二用法

这里将一一介绍我们使用中用到的方法
Length
这是字符串的一个属性，类似于数组的下标，例如
var str =’HELLO’;
这时想取到字符串指定位置的一个字符，那就是： Str[指定位置的下标]，str[0],str[1],str[2].
但是这里你要是想改变字符串指定位置的字符怎么办呢？
Str[0]=’P’;
这样可以改变字符串第一个位置的字符吗？答案是否定的，那么该用什么方式去实现这种需求呢？下面会讲到。
CharAt
获取字符串中的指定下标字符，不操作原字符串，返回获取到的字符。
用法：
var str1 = 'hello friends!'
str1.charAt(0);
console.log(str1.charAt(2));/l
console.log(str1);//hello friends!
charCodeAt
获取指定下标字符的ASCII码，返回字符的ASCII码，不操作原字符串。
用法：
var str1 = 'hello friends!'
console.log(str1.charCodeAt(5));//32
console.log(str1);//hello friends!
fromCharCode
获取指定编码的字符。
用法：console.log(String.fromCharCode(104));//h
indexOf
查找字符串中指定字符第一次出现的位置，返回指定字符出现的下标，如果没找到指定字符串，则返回-1，不改变原字符串。（从第一个位置开始查找）
用法：
var str1 = 'hello friends!'
console.log(str1.indexOf('e'));//1
console.log(str1.indexOf('k'));//-1
console.log(str1);//hello friends!
lastIndexOf
查找字符串中指定字符最后一次出现的位置，返回指定字符出现的下标，如果没找到指定字符串，则返回-1，不改变原字符串。（从第一个位置开始查找）
用法：
var str1 = 'hello friends!'
console.log(str1.lastIndexOf('e'));//9
console.log(str1.lastIndexOf('k'));//-1
console.log(str1);//hello friends!
replace
将字符串中的一部分替换为指定部分或者字符，返回新的字符串，不改变原字符串。
用法：
var str1 = 'hello friends!'
console.log(str1.replace('hello','hi'));//hi friends!
console.log(str1);//hello friends!
substring
从指定位置开始截取字符串，不包括结束的下标，返回新的字符串，不改变原字符串，
当开始下标大于结束下标时，就返回来截取，
当结束下标不写时，就会从起始下标截完，
当结束下标为负数时，从0开始截取，
当参数只有一个且为负数时，就截取完。
用法：
var str1 = 'hello friends!'
console.log(str1.substring(0,3));//hel
console.log(str1.substring(3,0));//hel
console.log(str1.substring(3));//lo friends!
console.log(str1.substring(-3));//hello friends!
console.log(str1);//hello friends!
substr
从指定位置开始截取指定长度的字符串，返回截取的字符串，不改变原字符串，
当长度为0或者负数时，会返回一个空字符串，
当长度不写时，从开始位置截取完。
用法：
var str1 = 'hello friends!'
console.log(str1.substr(0,3));//hel
console.log(str1.substr(3,0));//空
console.log(str1.substr(3));//lo friends!
console.log(str1);//hello friends!
split
根据指定分隔符将字符串分割为数组，不改变原字符串，返回分割成的数组。
用法：
var str1 = 'hello friends!'
console.log(str1.split(' '));//["hello", "friends!"]
console.log(str1.split('e'));//["h", "llo fri", "nds!"]
console.log(str1);//hello friends!
toLowerCase
将字符串转为小写，不操作原字符串，返回转换后的字符串。
用法：
var str1 = 'hEllo fRiends!'
console.log(str1.toLowerCase());//hello friends!
console.log(str1);//hEllo fRiends!
toUpperCase
将字符转换为大写，不操作原字符串，返回转换后的字符串。
用法：
var str1 = 'hEl4lo fRie3nds!'
console.log(str1.toUpperCase());//HEL4LO FRIE3NDS!
console.log(str1);//hEl4lo fRie3nds!
trim
去掉字符串中的前后空格，返回修改后的字符串，不改变原字符串。
用法：var str1 = ' hello friends! '
console.log(str1.trim());//'hello friends!'
console.log(str1)' hello friends! '
trimLeft
去掉字符串中的前空格，返回修改后的字符串，不改变原字符串。 用法：
var str1 = ' hello friends! '
console.log(str1.trim());//'hello friends !'
console.log(str1)' hello friends! '
trimRight
去掉字符串中的后空格，返回修改后的字符串，不改变原字符串。
用法：
var str1 = ' hello friends! '
console.log(str1.trim());//' hello friends!'
console.log(str1)' hello friends!



补充：在上面讲到了字符串中的字符查找和字符替换，这里就有一些实践的问题了，在字符串造作中会经常用到将字符串中的特定字符全部替换为指定字符，并且是全部替换，那就需要下面的这几种方法的配合使用了。 1)var reg=new RegExp("我们","g"); //创建正则RegExp对象
var stringObj="大家是幸福的，大家是快乐的";
var newstr=stringObj.replace(reg,"大家");
alert(newstr);
2)var str = “JavaScript预压是强大的计算机预压”;
alert(str.replace(“预压”,”语言“));//只替换最后一个
alert(str.replace(/预压/g,”语言“)) //全部替换【正则】

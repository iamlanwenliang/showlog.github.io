字符串的操作方法

在这篇里面会详细的讲到字符串相关的用法，注意事项等等

一、定义

字符串就是用单引号或者双引号括起来的字符，其中的字符可以是任何字符， 声明有两种方式：
var str = new String();
(效果等于str = {})这种方式严格的来说也可以，但是平时不建议使用，因为使用new出来的数据类型是object，但是这种声明出来的是一个类似于数组的字符串，将字符串按照length分割开来分别存在对象里面了。 Var str = ‘’;
这是常用的字符串声明方式，这种声明出来数据类型就是string的，也是比较正常的使用方式；
①　如果单引号中还包括单引号或者双引号里面还包括双引号，其中包涵的单引号或者双引号也是字符内容的话，一般可以用反斜杠（\）来表示，位置在特殊符号前面。这里要提醒的一点是字符串里面如果是ascII编码的话，在输出时浏览器会自动解析为相对应的字符。
②　这里介绍常用的两个小技巧，不过这是es6标准，如果你的浏览器不支持的话这里就需要更新到支持es6的最新版或者在支持es6的浏览器上调试。第一个就是怎么让字符串换行呢？我们一般第一想法都是在想要换行的地方添加\n就可以换行了，今天在这里介绍的是另一种es6标准里面的，用反引号括起来，反引号在键盘的esc键的下方第一个，字符串用反引号括起来，你在书写的时候字符串里面有几个换行，在显示时就显示几个换行
例如：`你的
我的` 那么字符串输出的时候也就是这样输出的。
第二个是字符串拼接，这个我相信在平时使用中大家是很常用到的，基本上大家现在的方式都是使用加号（+）把多个字符串连起来，今天介绍的是es6新增的一种模板字符串，
例如：var man = ‘那个男人’;
var using = ‘看起来’;
var strong = ‘很强壮’;
var talk = ‘${man}${using}${strong}’;
这样就拼起来了，是不是很方便呢

二用法

这里将一一介绍我们使用中用到的方法
1、Length
这是字符串的一个属性，类似于数组的下标，例如
var str =’HELLO’;
这时想取到字符串指定位置的一个字符，那就是： Str[指定位置的下标]，str[0],str[1],str[2].
但是这里你要是想改变字符串指定位置的字符怎么办呢？
Str[0]=’P’;
这样可以改变字符串第一个位置的字符吗？答案是否定的，那么该用什么方式去实现这种需求呢？下面会讲到。
2、CharAt
获取字符串中的指定下标字符，不操作原字符串，返回获取到的字符。
用法：
var str1 = 'hello friends!'
str1.charAt(0);
console.log(str1.charAt(2));/l
console.log(str1);//hello friends!
3、charCodeAt
获取指定下标字符的ASCII码，返回字符的ASCII码，不操作原字符串。
用法：
var str1 = 'hello friends!'
console.log(str1.charCodeAt(5));//32
console.log(str1);//hello friends!
4、fromCharCode
获取指定编码的字符。
用法：console.log(String.fromCharCode(104));//h
5、indexOf
查找字符串中指定字符第一次出现的位置，返回指定字符出现的下标，如果没找到指定字符串，则返回-1，不改变原字符串。（从第一个位置开始查找）
用法：
var str1 = 'hello friends!'
console.log(str1.indexOf('e'));//1
console.log(str1.indexOf('k'));//-1
console.log(str1);//hello friends!
6、lastIndexOf
查找字符串中指定字符最后一次出现的位置，返回指定字符出现的下标，如果没找到指定字符串，则返回-1，不改变原字符串。（从第一个位置开始查找）
用法：
var str1 = 'hello friends!'
console.log(str1.lastIndexOf('e'));//9
console.log(str1.lastIndexOf('k'));//-1
console.log(str1);//hello friends!
7、replace
将字符串中的一部分替换为指定部分或者字符，返回新的字符串，不改变原字符串。
用法：
var str1 = 'hello friends!'
console.log(str1.replace('hello','hi'));//hi friends!
console.log(str1);//hello friends!
8、substring
从指定位置开始截取字符串，不包括结束的下标，返回新的字符串，不改变原字符串，
当开始下标大于结束下标时，就返回来截取，
当结束下标不写时，就会从起始下标截完，
当结束下标为负数时，从0开始截取，
当参数只有一个且为负数时，就截取完。
用法：
var str1 = 'hello friends!'
console.log(str1.substring(0,3));//hel
console.log(str1.substring(3,0));//hel
console.log(str1.substring(3));//lo friends!
console.log(str1.substring(-3));//hello friends!
console.log(str1);//hello friends!
9、substr
从指定位置开始截取指定长度的字符串，返回截取的字符串，不改变原字符串，
当长度为0或者负数时，会返回一个空字符串，
当长度不写时，从开始位置截取完。
用法：
var str1 = 'hello friends!'
console.log(str1.substr(0,3));//hel
console.log(str1.substr(3,0));//空
console.log(str1.substr(3));//lo friends!
console.log(str1);//hello friends!
10、split
根据指定分隔符将字符串分割为数组，不改变原字符串，返回分割成的数组。
用法：
var str1 = 'hello friends!'
console.log(str1.split(' '));//["hello", "friends!"]
console.log(str1.split('e'));//["h", "llo fri", "nds!"]
console.log(str1);//hello friends!
11、toLowerCase
将字符串转为小写，不操作原字符串，返回转换后的字符串。
用法：
var str1 = 'hEllo fRiends!'
console.log(str1.toLowerCase());//hello friends!
console.log(str1);//hEllo fRiends!
12、toUpperCase
将字符转换为大写，不操作原字符串，返回转换后的字符串。
用法：
var str1 = 'hEl4lo fRie3nds!'
console.log(str1.toUpperCase());//HEL4LO FRIE3NDS!
console.log(str1);//hEl4lo fRie3nds!
13、trim
去掉字符串中的前后空格，返回修改后的字符串，不改变原字符串。
用法：var str1 = ' hello friends! '
console.log(str1.trim());//'hello friends!'
console.log(str1)' hello friends! '
14、trimLeft
去掉字符串中的前空格，返回修改后的字符串，不改变原字符串。 用法：
var str1 = ' hello friends! '
console.log(str1.trim());//'hello friends !'
console.log(str1)' hello friends! '
15、trimRight
去掉字符串中的后空格，返回修改后的字符串，不改变原字符串。
用法：
var str1 = ' hello friends! '
console.log(str1.trim());//' hello friends!'
console.log(str1)' hello friends!
16、fromCharCode
fromCharCode() 方法返回使用指定的Unicode值序列创建的字符串。
用法：
String.fromCharCode(num1, ..., numN)
num1, ..., numN
一组序列数字，表示 Unicode 值。
该方法返回一个字符串，而不是一个 String 对象。
例如：
String.fromCharCode(65,66,67)//ABC
17、fromCodePoint
fromCodePoint() 静态方法返回使用指定的代码点序列创建的字符串。
用法：
String.fromCodePoint(num1[, ...[, numN]])
num1, ..., numN
一串 Unicode 编码。
RangeError
如果传入无效的 Unicode 编码，将会抛出一个RangeError (例如： "RangeError: NaN is not a valid code point")。
例如：
String.fromCodePoint(42); // "*"
String.fromCodePoint(65, 90); // "AZ"
String.fromCodePoint(0x404); // "\u0404"
String.fromCodePoint(0x2F804); // "\uD87E\uDC04"
String.fromCodePoint(194564); // "\uD87E\uDC04"
18、endsWith
endsWith()方法用来判断当前字符串是否是以另外一个给定的子字符串“结尾”的，根据判断结果返回 true 或 false。（ES6规范，使用注意兼容性）
用法：
str.endsWith(searchString [, position]);
searchString
要搜索的子字符串。
position
在 str 中搜索 searchString 的结束位置，默认值为 str.length，也就是真正的字符串结尾处。
例如：
var str = "To be, or not to be, that is the question.";
alert( str.endsWith("question.") ); // true
alert( str.endsWith("to be") ); // false
alert( str.endsWith("to be", 19) ); // true
alert( str.endsWith("To be", 5) ); // true
19、includes
includes() 方法用于判断一个字符串是否包含在另一个字符串中，根据情况返回true或false。
用法：
str.includes(searchString[, position])
searchString
要在此字符串中搜索的字符串。
position
可选。从当前字符串的哪个索引位置开始搜寻子字符串；默认值为0。
如果当前字符串包含被搜寻的字符串，就返回true；否则，返回false。
includes() 方法是区分大小写的。
例如：
var str = 'To be, or not to be, that is the question.';
console.log(str.includes('To be')); // true
console.log(str.includes('question')); // true
console.log(str.includes('nonexistent')); // false
console.log(str.includes('To be', 1)); // false
console.log(str.includes('TO BE')); // false
20、match
当一个字符串与一个正则表达式匹配时， match()方法检索匹配项。
语法：
str.match(regexp);
regexp
一个正则表达式对象。如果传入一个非正则表达式对象，则会隐式地使用 new RegExp(obj) 将其转换为一个 RegExp 。如果你未提供任何参数，直接使用 match() ，那么你会得到一个包含空字符串的 Array ：[""] 。
array
如果字符串匹配到了表达式，会返回一个数组，数组的第一项是进行匹配完整的字符串，之后的项是用圆括号捕获的结果。如果没有匹配到，返回null
21、padEnd
padEnd() 方法会用一个字符串填充当前字符串（如果需要的话则重复填充），返回填充后达到指定长度的字符串。从当前字符串的末尾（右侧）开始填充。
语法：
str.padEnd(targetLength [, padString])
targetLength
当前字符串需要填充到的目标长度。如果这个数值小于当前字符串的长度，则返回当前字符串本身。
padString 可选
填充字符串。如果字符串太长，使填充后的字符串长度超过了目标长度，则只保留最左侧的部分，其他部分会被截断。此参数的缺省值为 " "（U+0020）。
在原字符串末尾填充指定的填充字符串直到目标长度所形成的新字符串。
例如：
'abc'.padEnd(10); // "abc "
'abc'.padEnd(10, "foo"); // "abcfoofoof"
'abc'.padEnd(6, "123456"); // "abc123"
'abc'.padEnd(1); // "abc"
22、padStart
padStart() 方法用另一个字符串填充当前字符串(重复，如果需要的话)，以便产生的字符串达到给定的长度。填充从当前字符串的开始(左侧)应用的。
用法：
str.padStart(targetLength [, padString])
targetLength
当前字符串需要填充到的目标长度。如果这个数值小于当前字符串的长度，则返回当前字符串本身。
padString 可选
填充字符串。如果字符串太长，使填充后的字符串长度超过了目标长度，则只保留最左侧的部分，其他部分会被截断。此参数的缺省值为 " "（U+0020）。
在原字符串开头填充指定的填充字符串直到目标长度所形成的新字符串。
例如：
'abc'.padStart(10); // " abc"
'abc'.padStart(10, "foo"); // "foofoofabc"
'abc'.padStart(6,"123465"); // "123abc"
'abc'.padStart(8, "0"); // "00000abc"
'abc'.padStart(1); // "abc"
23、repeat
repeat() 构造并返回一个新字符串，该字符串包含被连接在一起的指定数量的字符串的副本。
/**
* str: String
* count: Number
*/
let resultString = str.repeat(count);
count
介于0和正无穷大之间的整数 : [0, +∞) 。表示在新构造的字符串中重复了多少遍原字符串。
包含指定字符串的指定数量副本的新字符串。
例如：
"abc".repeat(-1) // RangeError: repeat count must be positive and less than inifinity
"abc".repeat(0) // ""
"abc".repeat(1) // "abc"
"abc".repeat(2) // "abcabc"
"abc".repeat(3.5) // "abcabcabc" 参数count将会被自动转换成整数.
"abc".repeat(1/0) // RangeError: repeat count must be positive and less than inifinity
({toString : () => "abc", repeat : String.prototype.repeat}).repeat(2) //"abcabc",repeat是一个通用方法,也就是它的调用者可以不是一个字符串对象.
24、search
search() 方法执行正则表达式和 String对象之间的一个搜索匹配。
用法：
str.search(regexp)
regexp
一个正则表达式（regular expression）对象。如果传入一个非正则表达式对象，则会使用 new RegExp(obj) 隐式地将其转换为正则表达式对象。
如果匹配成功，则 search() 返回正则表达式在字符串中首次匹配项的索引。否则，返回 -1。
当你想要知道字符串中是否存在某个模式（pattern）时可使用 search，类似于正则表达式的 test 方法。当要了解更多匹配信息时，可使用 match（会更慢），该方法类似于正则表达式的 exec 方法。
例如：
function testinput(re, str){
var midstring;
if (str.search(re) != -1){
midstring = " contains ";
} else {
midstring = " does not contain ";
}
console.log (str + midstring + re);
}
25、startsWith
startsWith()方法用来判断当前字符串是否是以另外一个给定的子字符串“开头”的，根据判断结果返回 true 或 false。（ES6新规范，使用注意兼容）
用法：
str.startsWith(searchString [, position]);
searchString
要搜索的子字符串。
position
在 str 中搜索 searchString 的开始位置，默认值为 0，也就是真正的字符串开头处。
例如：
var str = "To be, or not to be, that is the question.";
alert(str.startsWith("To be")); // true
alert(str.startsWith("not to be")); // false
alert(str.startsWith("not to be", 10)); // true

语法

Javascript 的语法和许多计算机语言很类似，如果学者在学习JavaScript之前有其他语言的基础，比如c++/c/java等等，那么JavaScript学习起来就很容易了，以为大部分语法都很类似。
1、JavaScript的每个语句以‘;’结束，语句块用‘{}’，但是在JavaScript中并不是强制要求说是在每个语句的结尾必须加‘;’,只要代码编写中没有语法错误，浏览器的执行JavaScript代码的引擎会在执行的时候会自动在语句末尾加上‘;’。
2、在JavaScript里面变量 是用var去声明的，在es6中还与几种声明方式，let和const，这两个后面会详细讲到。
3、基本数据类型有：number，object，undefined，null，Boolean，string这些都是基本数据类型，还有两种比较常用的数据类型：array和function，这些都是属于引用类型。
4、再写嵌套语句块的时候建议增加缩进，虽然JavaScript对嵌套层级没有限制，但是不建议过多的嵌套，这样会增减代码的可读性，如果这种情况建议把部分代码抽出来单独作为一个功能函数来调用，不建议一行写多句代码，有缩进的时候有助于我们对代码层次的理解和后期维护，

数据类型

1、number
在JavaScript中浮点数和整数都是number类型的，数据采用IEEE754标准定义的64位浮点格式， 常见的数据类型大概有：整数，浮点数（带有小数的），用科学计数法表示的（1.02e5=1.02x100000）,负数，NaN；此类情况经常在数据转换或者运算的时候出现，表示not a number，最后一种是infinity，这是数值超过了JavaScript的number类型所能表示的最大数时就表示为infinity（JavaScript中number所能表示的最大数范围是：整数-253---253也就是16位，超过的全部显示为0，浮点数±1.7976931348623157 × 10308-----±5 × 10−324）， Number数据可以做四则运算；+,-,*,/,%,>,<,=,==,===,<=,>=.
2、string
JavaScript中的字符串是用单引号或者双引号括起来的字符，还可以引号里面直接写字符的ascII编码，小tips：多行字符串，如果觉得\n写起来麻烦，那么还有一种方式，在es6里面新增了一种便是方法，就是用反引号，用反引号括起来的字符串了面可以换行展示，（反引号在数字键1的左边），字符串可以进行四则运算中的加运算，其他的运算不能进行。
3、Boolean
布尔值的取值只有两种即：true，和false，白话的说就是存在和不存在，也可以去判断一个运算式成立或者不成立，比如a>b，成立或者不成立，多个判断或者存在不存在的时候总体就有两种运算符‘||’和“&&”,前者就是意思是什么或者什么，也就是说在多个判断中，只要有一个判断是成立的，那么整个判断就是成立的，后者刚好相反，同样的在多个判断中，只要有一个不成立，那么整个判断就是不成立的。
4、Object
在JavaScript中的object通常叫做对象，就是又一组键和值组成的无序集合，对象的键都是字符串类型的，值可以是任意的数据类型，例如： Obj={ Nema:’小明’, Sex:’男’, Age:’19’, Height:’187’ } 其中键值可以是任何的数据类型，在引用的时候，比如要引用‘小明’，就是obj.Name.
5、null和undefined
null表示一个空的值，数据类型是object，null和0不同，0是一个数值，undefined表示未定义，也就是没有定义数据类型，通常在给变量值的时候就给变量赋予了数据类型，但是undefined就是你声明了这个变量，但是没有给变量赋值，也就是没有给变量数据类型。至于null、undefined，0，’’，[]，{}，的区别和识别后面会详细讲到。
6、Array
这是数组类型，也就是说是一组按顺序排列的集合，集合中的每个元素的值也可以是任意数据类型的数据，一般我们用到的最多就是一维数组和二维数组，虽然对数组的维度没有限制，但是严格的来说数组的数据维度越多那么我们操作数据和读取数据的复杂度和难度就会相对提升，所以一般情况下我们通常在使用中，一维数组基本就可以满足我们的需求了，二维数组几乎也很少用到。
7、Function
Function，这是一种很特殊的数据类型，这是一个小的代码集合用于处理特殊的加工，这样来说吧，function就是具有特定功能的代码集合，打个比方，就是一个小的加工厂，材料从这个加工厂进来经过这个小加工厂的处理输出特定的成品或者半成品产品。这个后面也会详细讲到。
